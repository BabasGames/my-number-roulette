## My Number Roulette

A simple number roulette game built with React.

## Installation

1. Clone the repository:
bash
   git clone https://github.com/username/my-number-roulette.git
   cd my-number-roulette
   
2. Install dependencies:
bash
   npm install
   
3. Start the application:
bash
   npm start
